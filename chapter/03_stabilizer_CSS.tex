\chapter{スタビライザー形式とCSS符号}
これまではShor符号の具体的な構成を通して、
量子誤り訂正符号の基本的なアイデアを紹介してきた。
このアプローチは情報を保護する量子ビットを具体的にどのような形でエンコードするかを先に定め、
その後にエンコードされた状態を壊さずに誤りを検出するための測定演算子を見つけるというものであった。
しかし、より高い誤り耐性などの性能を求めて複雑な符号を構築しようとする際、
この方法は非常に煩雑になり、いずれ限界を迎えることになる。
そこで本章では量子誤り訂正符号の構成法を考えるにあたって基本となる
スタビライザー形式\cite{Gottesman1997-ep,Daniel-Gottesman1999-eh,Bullock2007-jl}と
CSS 符号\cite{Calderbank1996-si,Steane1996-is}
を見ていく。
これはまずシンドローム測定を与える演算子を定め、
符号空間をそれらの共通固有空間として構成する。
CSS符号はスタビライザー形式のうち、
シフトエラーと位相エラーに対応する検査行列からシンドローム測定する演算子を構成するクラスである。
これにより、符号の構成が容易になるだけではなく符号の物理的な性質を解析するのにも非常に便利な枠組みが得られる。
