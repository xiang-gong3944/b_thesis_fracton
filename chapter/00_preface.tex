\section*{歴史}
多体系における中心的な課題の1つは、相の分類、およびその性質と起源を理解することである。
最も古典的な枠組みである熱力学は、マクロな変数によって相を記述するが、系がなぜその相へ至るのかという微視的機構までは説明し得なかった。
その後、統計力学の発展により、微視的自由度から熱力学量を導く道筋が整備され、相転移を相関やゆらぎの観点から捉えることが可能となった。
しかし、臨界点近傍の現象や相の多様性を系統立てて理解するためには、より構造的な言語が必要とされていた。

この課題に対し、現代的な相分類の基礎を与えたのが Landau 理論である。
Landau の枠組みでは、相は自発的対称性の破れによって特徴づけられ、局所的な秩序変数が非自明な期待値を取ることで相の区別が可能になる。
この対称性と秩序変数に基づく分類は多体系物理の標準的な指針として確立された。

しかしその後、分数量子ホール効果やSPT相といった強相関電子系の研究、さらに場の量子論におけるアノマリーやゲージ理論の理解が進むにつれて、
局所秩序変数や対称性の破れだけでは記述できない相の存在が明らかになった。
これらの相は系の大域的な性質や境界自由度、場合によっては長距離エンタングルメントに由来する頑健な性質を示し、
微視的な詳細によらないトポロジーによって特徴づけられることからトポロジカル相と呼ばれる。
現在、これは現代物理学の最重要テーマの1つとして活発に研究されている。

トポロジカル秩序相の顕著な性質として、非自明な統計性を持つ準粒子(エニオン)の存在が挙げられる。
エニオンは、互いの位置を交換(Braiding)することにより波動関数が位相を獲得したり、異なる粒子の重ね合わせ状態へと変化したりする性質を持つ。
これらは分数量子ホール系やスピン液体における低エネルギー有効模型として理解されている。

こうしたトポロジカル秩序の性質は、量子情報の観点からも極めて重要である。
トポロジカル秩序相では、基底状態の縮退度が空間のトポロジーに依存し、十分大きな系では局所演算子によって基底状態同士を区別できない。
これは、量子情報が局所自由度ではなく、系全体の非局所的自由度に分散して符号化されていることを意味する。
したがって、局所的なエラーは論理状態である基底状態間を遷移させることができず、単に準粒子を生成する操作として記述される。
この準粒子の配置を測定し(シンドローム測定)、適切な操作によって消滅させる(訂正操作)ことで情報を保護できるため、
エニオンの基礎研究は量子誤り訂正符号(QEC)への応用という観点からも強く動機づけられている。

一方で、QECの側からの研究動機は必ずしも「エニオンや対称性」のみを出発点とするわけではない。
むしろ、局所的に検査可能な制約(スタビライザー)から、可解な量子スピン模型をいかに系統的に構成し、
より高性能な符号へ拡張するかという構成論的な関心が主導してきた側面がある。

例えば可解な局所ハミルトニアンを系統的に構成する手法として、Tillich-Zémor による Hypergraph Product (HGP)符号が知られている。
HGP符号は2つの古典線形符号から量子誤り訂正符号を構成する手法であり、最も基本的な例として、2つの 繰り返し符号の HGP が Toric Code を与えることが知られている。
一般に誤り訂正符号を古典スピン模型と読み替えることができる。これより繰り返し符号を読み替えると1次元イジング模型となることからイジング模型を2つ組み合わせるとToric codeになるとも言える。

このような構成的アプローチの過程で、新たに得られた符号や模型が従来の2次元エニオン像とは異なる特異な励起を示すこともあるということがしばしあるがあると判明した。
通常の2次元エニオンが平面内を自由に動ける点状励起であるのに対し、特定の3次元局所可解模型には、
単独では動けない励起 (Fracton)、直線上にしか動けない励起 (Lineon)、平面内にしか動けない励起 (Planon) など、低次元的な (Sub-dimensional)運動制約を持つ励起が自然に現れる。
これらは格子上の局所構造や制約条件と深く結びついており、単純なトポロジーだけでは理解しにくい新しい相「フラクトン相」として、近年体系的な整理が進められている。

フラクトン相の理解には複数のアプローチが存在するが、その中でも「多極子保存則」でのアプローチに焦点を当てる。
連続体の有効理論として現れる高階対称テンソルゲージ理論では、Gauss 則が全電荷に加えて双極子モーメント、さらには高次の多極子の保存を要請する。
特に電荷と双極子が共に保存される系では、孤立した電荷を局所的に移動させる操作は双極子モーメントを変化させてしまうため禁止され、結果として点状励起の運動が強く制限される。
これにより、フラクトン的な振る舞いが対称性の観点から自然に理解される。

このような保存則は、静的な励起構造だけでなく、動力学や熱化の過程にも本質的な影響を与える。
双極子保存を持つ系では、輸送や緩和が通常の拡散とは異なるスケーリングを示すことが議論された。
さらに、保存則と局所制約の組み合わせはヒルベルト空間の有効な分断(Fragmentation)を引き起こし、熱化が部分空間内に制限された形でしか起こらない状況や、
量子スカーに類似した非熱的な定常状態(Dynamical Scars)の出現をもたらすことが知られている。
これらは、従来の統計力学が前提としてきた「典型的な熱化」の直観が、多極子保存のような強い制約下では大きく修正されることを示唆している。


\section*{概要(研究)}
量子誤り訂正の文脈では、Toric codeの課題の1つはエニオンがエネルギーコスト無しに自由に動けることがある。
この性質のため熱揺らぎによって励起が拡散しやすく、有限温度では自己修正メモリになりえず符号へ常に訂正操作を加え続けなければならない。
この性質の由来はどこからきているかというと、HGP前のイジング模型からきている。
イジング模型において「ドメインウォール(励起)が自由に動ける」という性質をもち、これが HGP を経ても残っているため、
Toric Code における「エニオンが自由に動ける」という性質を生み出している。

この観点に基づくと、HGP が生み出す2次元トポロジカル符号の「励起の可動性」はもとの古典スピン模型が持つ 局所制約の性質と関連することが期待される。
そこで本研究では、HGPの元となるFracton的な制約を持ったグラフラプラシアン由来のスピン模型を考えそれらのHGPで得られる量子誤り訂正符号上の励起の持つ物理を調べる。

具体的には1次元鎖のグラフラプラシアン由来の古典スピン模型である

$$H = -\sum_i Z_{i-1}Z^{\dagger 2}_{i} Z_{i+1}$$

を HGP して得られる量子スピン模型の励起の性質を調べる。
ここで、Zは $\mathbb{Z}_N$ におけるパウリ演算子である。
この模型はフラクトン的な双極子保存則をもち、励起した粒子の運動に強い制約が課すことを知られている。
古典スピン模型の時点で多くの性質が知られているこの模型であるが、これ同士をHGPして得られる量子誤り訂正符号についての性質は調べられていないように見える。
1次元系から2次元系になったことにより、準粒子の統計性を調べることが可能になったり、
ドメインウォールがエネルギーバリアを持つことが期待され自己修正メモリになりうるのではないかというので、
量子誤り訂正符号・量子多体系双方として非自明な結果が期待できる。
